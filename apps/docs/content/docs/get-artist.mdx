### Endpoint

```
GET https://api.stats.skylerx.ir/v1/artists/{artistId}
```


## Input

The `artistId` has to be based on the the Spotify Artist Id. You can get this id using the [SpotifyAPI](https://developer.spotify.com/documentation/web-api/reference/#/operations/get-an-artist) or by using the [Spotify Web Player](https://open.spotify.com/).

```ts
{
  artistId: string;
}
```
