### Endpoint

```
GET https://api.stats.skylerx.ir/v1/tracks/{trackId}
```


## Input

The `trackId` has to be based on the the Spotify Track Id. You can get this id using the [SpotifyAPI](https://developer.spotify.com/documentation/web-api/reference/get-track) or by using the [Spotify Web Player](https://open.spotify.com/).

```ts
{
  trackId: string;
}
```
